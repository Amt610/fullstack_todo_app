About Todo Application

A Todo Application helps you organize your work by allowing you to create, manage, update, and complete tasks with ease. Whether you're planning your daily schedule, managing office work, tracking assignments, or organizing personal goals, a Todo App ensures that nothing gets overlooked. Its clean interface and simple workflow make task management quick and effective.

Features

📝 Create Tasks

Easily add new tasks and organize your daily activities.

📋 View All Tasks

Display all your tasks in a clean and organized dashboard.

✏️ Edit Tasks

Update task details whenever your plans change.

✅ Mark as Completed

Track completed work and stay focused on pending tasks.

🗑 Delete Tasks

Remove unnecessary or completed tasks with one click.

💾 H2 Database

Store task information securely using the built-in H2 database.

⚡ Spring Boot

Built using Spring Boot, Spring Data JPA, and Thymeleaf for rapid development.

📱 Responsive Design

Works smoothly on desktops, tablets, and mobile devices.

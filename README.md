# 📝 Spring Boot Todo Application

A modern **Todo Management Application** built using **Spring Boot**, **Spring Data JPA**, **Thymeleaf**, and **H2 Database**. This project allows users to create, update, manage, and delete daily tasks through a clean and responsive web interface.

---

## 🚀 Features

* ✅ Create a new Todo
* ✅ View all Todos
* ✅ Edit existing Todos
* ✅ Mark Todos as Completed
* ✅ Delete Todos
* ✅ Responsive User Interface
* ✅ H2 In-Memory Database
* ✅ Spring Boot MVC Architecture
* ✅ Thymeleaf Template Engine
* ✅ Spring Data JPA

---

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/spring-boot-todo-app.git
```

### Navigate to the project

```bash
cd spring-boot-todo-app
```

### Run the project

```bash
mvn spring-boot:run
```
